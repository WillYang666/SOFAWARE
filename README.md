# GIF制作

- ScreenToGIF(软件）

[ScreenToGif](https://www.screentogif.com)

- GifCam（软件）

### 功能特性

录制过程中，可以随意改变窗口大小、位置。

录制范围内内容无变化时，不增加新帧而只增加延时，减小文件尺寸。

录制：3种帧速率(FPS)/全屏模式/可设置是否捕获鼠标

编辑：删除帧/修改帧延时/添加文字

保存：六种色彩质量保存方案

绿色单文件，即点即用
